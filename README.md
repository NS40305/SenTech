# SEN TECH Website

Production static site for SEN TECH (sen-tech.com).

## Stack
Astro 4 + TypeScript + CSS custom properties. Bilingual EN / 繁中.

## Local development
```bash
npm install
npm run dev    # http://localhost:4321
npm run build  # outputs to dist/
npm run test   # vitest
```

## Deployment
Cloudflare Pages via Git integration.
